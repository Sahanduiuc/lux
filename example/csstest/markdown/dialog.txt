
Dialog
=================

A dialog is created via the `dialog` widget.

<div class="lux-example">

<div title="Hello" class="dialog block-control" data-width="300">
Hello, I'm a simple dialog with an Hello header.
</div>

</div>

Collapsed dialog
-------------------------
To create a collapsable dialog already collapsed set the `collapsable` parameter
to `true`.

<div class="lux-example">
<div id="example3" class="block-control"></div>

<script type="text/javascript">

lux.web.ready(["lorem"], function () {
    $('#example3').lorem({words:20}).dialog({
        title: 'A collapsed dialog',
        skin:'primary',
        width: 400,
        collapsable:true,
        collapsed:true
    });
});

</script>
</div>

    $('#example3').dialog({
        title: 'A collapsed dialog',
        skin:'primary',
        width: 400,
        collapsable:true,
        collapsed:true
    });


</script>
